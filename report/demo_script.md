# WeatherIQ Demo Script (1–2 Minutes)

Hi, this is my WeatherIQ project for the PM Accelerator Weather Trend Forecasting assessment.

I used the Kaggle Global Weather Repository dataset stored locally as `data/GlobalWeatherRepository.csv`. No external weather API or API key is used.

On the backend, I built a modular FastAPI service with endpoints for dataset summary, cleaning summary, EDA trends, forecasting, advanced analysis, spatial summaries, and final insights.

On the frontend, I built a premium dashboard with pages for Overview, Dataset, Data Cleaning, EDA, Forecasting, Advanced Analysis, Spatial Insights, PM Accelerator Mission, and Final Insights.

In EDA, I show temperature, precipitation, humidity, and wind trends, plus city-level comparisons and correlation analysis.

In Forecasting, I compare Moving Average, Linear Regression, Random Forest, Gradient Boosting, and an Ensemble model using MAE, RMSE, MAPE, and R², then display actual vs predicted and future forecast charts.

In Advanced Analysis, I show Isolation Forest anomaly detection, air quality relationships, feature importance, and climate patterns.

In Spatial Insights, I visualize city points globally and summarize country-level weather and air quality risk.

I also included a dedicated PM Accelerator Mission page and a final insights page with recommendations and a full assessment completion checklist.

This project satisfies the assessment requirements end-to-end.
